Как сделать сайт

    

    Что используем для создания сайта

    

    Чаще всего начинающие дизайнеры начинают свою работу с поиска подходящей программы для веб дизайна. Обычно выбирают то, что можно скачать бесплатно.

    В любом случае нужно иметь два графических редактора - векторный и растровый. Векторный редактор используется для создания макета дизайна сайта, увязки расположения элементов, правки.

    Растровый графический редакто используется для выделения из макета ключевых графических элементов в отдельные загружаемые файлы, которые используются в HTML коде. Заполнение пространств веб страницы цветом, позиционирование элементов дизайна, расстояния - задаются в HTML коде.

    В качестве векторного редактора лучше всего использовать Corel Draw, пусть даже самых ранних версий. Этого достаточно. В качестве растрового графического редактора лучше всего - Adobe Photoshop ранних версий. Ранние версии редакторов компактны, менее требовательны к "железу" и могут быть куплены с лицензионными ключами за малые деньги.

    HTML код можно писать в notepad.exe - простом текстовом редакторе. Если вы сосздаете простые веб страницы, это позволит вам освоить HTML и получить очень простой и понятный код.
